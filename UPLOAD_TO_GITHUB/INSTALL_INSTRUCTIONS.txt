===============================================
HOW TO ADD THIS TO YOUR PROJECT
===============================================

EASIEST METHOD: Just drag and drop!

===============================================
METHOD 1: DRAG AND DROP (SIMPLEST)
===============================================

1. Open this folder (UPLOAD_TO_GITHUB)
2. Open your project folder in another window
3. Drag package.json from here to your project root
4. Replace the existing package.json
5. Drag android-build.sh to your project root
6. Done!

Then:
- Git add and commit
- Push to GitHub
- Update AppFlow settings (see CI_CD_QUICK_FIX.txt)

===============================================
METHOD 2: COPY COMMAND (TERMINAL)
===============================================

Open Terminal in your project folder:

cd /path/to/your/project

Then copy files:

cp UPLOAD_TO_GITHUB/package.json .
cp UPLOAD_TO_GITHUB/android-build.sh .
chmod +x android-build.sh

Optional (documentation):
cp UPLOAD_TO_GITHUB/*.txt .

Then commit:

git add package.json android-build.sh
git commit -m "Fix CI/CD Android build"
git push

===============================================
METHOD 3: GITHUB WEB (NO TERMINAL)
===============================================

For package.json:
1. Go to your repo on GitHub.com
2. Click on "package.json"
3. Click Edit button (pencil icon)
4. Select all and delete
5. Open package.json from this folder
6. Copy everything
7. Paste into GitHub editor
8. Scroll down, write commit message: "Fix CI/CD build"
9. Click "Commit changes"

For android-build.sh:
1. In your repo on GitHub.com
2. Click "Add file" â†’ "Upload files"
3. Drag android-build.sh from this folder
4. Write commit message: "Add build script"
5. Click "Commit changes"
