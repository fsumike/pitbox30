===============================================
HOW TO USE THIS KEYSTORE:
===============================================

When building release APK/AAB in Android Studio:

1. Build → Generate Signed Bundle / APK
2. Select "Android App Bundle" (or APK)
3. Click "Choose existing..."
4. Browse to keystore file location (see above)
5. Enter keystore password (see above)
6. Select key alias (see above)
7. Enter key password (see above)
8. Click Next → Finish
9. Wait for build to complete

Output location: android/app/build/outputs/bundle/release/

===============================================
RECOVERY PLAN:
===============================================

If keystore is lost:
1. You CANNOT update existing Google Play Store app
2. You must create NEW keystore
3. You must publish as NEW app (new package name)
4. Users must uninstall old app and install new one
5. Lose all reviews, ratings, and download counts

THIS IS WHY BACKUP IS CRITICAL!
