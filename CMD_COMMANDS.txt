COMMANDS TO RUN IN CMD/POWERSHELL:

1. BUILD FOR ANDROID:
   npm run build:android
   npm run cap:open:android
   (Opens Android Studio - click "Run" button)

2. BUILD FOR iOS:
   npm run build:ios
   npm run cap:open:ios
   (Opens Xcode - click "Play" button)

3. DEPLOY LIVE UPDATE (Capawesome):
   npm run capawesome:login
   npm run deploy:bundle

4. BUILD ON CAPAWESOME CLOUD:
   npm run capawesome:build:android
   npm run capawesome:build:ios

5. TEST IN BROWSER:
   npm run dev
   (Opens at http://localhost:5173)

========================================
WHAT EACH COMMAND DOES:
========================================

✅ npm run build:android
   - Builds your web app
   - Syncs to Android platform
   - Prepares for Android Studio

✅ npm run build:ios
   - Builds your web app
   - Syncs to iOS platform
   - Prepares for Xcode

✅ npm run deploy:bundle
   - Builds web app
   - Creates signed bundle
   - Uploads to Capawesome Cloud
   - Users get update automatically!
