üîí How Android Permissions Work

Step 1: Declare permissions in AndroidManifest.xml (‚úÖ Done!)

Step 2: Capacitor automatically requests permissions when you use a plugin

Step 3: Android shows a system dialog asking the user to allow or deny

Example: When you call Camera.getPhoto(), Android automatically shows "PIT-BOX wants to access your camera" with Allow/Deny buttons.
