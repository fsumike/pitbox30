Installation Instructions:

  1. Download the android-gradle-8.11.tar.gz file
  2. Extract it to your project root directory:
    tar -xzf android-gradle-8.11.tar.gz
  3. Open Android Studio and select "Open an Existing Project"
  4. Navigate to the android folder
  5. Let Gradle sync (it will download Gradle 8.11.1 automatically)
  6. Build your APK/AAB as normal!
