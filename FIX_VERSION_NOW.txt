STEP 1: Create Git Repository on GitHub
────────────────────────────────────────
1. Go to https://github.com/new
2. Create repository named: pitbox-app
3. Copy the URL (looks like: https://github.com/YOUR_USERNAME/pitbox-app.git)


STEP 2: Push Your Code to GitHub
────────────────────────────────────────
Run these commands in your project folder:

git init
git add .
git commit -m "PitBox v1.1.0 build 126"
git branch -M main
git remote add origin YOUR_GITHUB_URL_HERE
git push -u origin main


STEP 3: Connect Capawesome to Your GitHub
────────────────────────────────────────
1. Go to https://cloud.capawesome.io
2. Click on your PitBox app
3. Go to Settings → Git Integration
4. Connect your GitHub account
5. Select the pitbox-app repository
6. Set branch to "main"
7. Save


NOW BUILD:
────────────────────────────────────────
npm run capawesome:build:ios

   WHY THIS WAS HAPPENING
═══════════════════════════════════════════════════════════════

When you run: npm run capawesome:build:ios

Capawesome Cloud:
  1. Clones YOUR Git repository ← YOU DON'T HAVE ONE YET
  2. Reads capawesome.config.json
  3. Builds the app
  4. Uploads to App Store

Since you have NO Git repository:
  → Capawesome can't find your code
  → It uses default settings
  → Version stays at 1.0 (1)
