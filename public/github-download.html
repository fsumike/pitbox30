📋 Quick Start (Windows)

  1. Extract the downloaded file to C:\Users\YourName\Projects\
  2. Open folder and Shift+Right-click → "Open PowerShell window here"
  3. Verify: Run git status (should show "On branch main")
  4. Create token: Visit github.com/settings/tokens
  5. Create repo: Go to github.com/new → Name: pitbox-app
  6. Push: Run git push -u origin main (use token as password)
⚠️ Common Mistake: Don't run Git commands from C:\Windows\System32 - you must be inside the extracted project folder!
