📱 iOS Platform for PitBox

This package contains the complete Capacitor iOS platform needed for Capawesome Cloud builds.

⬇ Download iOS Platform (PitBox-iOS-Platform.tar.gz)
⚠️ Important: This iOS platform folder MUST be committed to your GitHub repository for Capawesome Cloud builds to work.

Installation Instructions

1
Download the file above

Save PitBox-iOS-Platform.tar.gz to your computer

2
Navigate to your project directory
cd /path/to/pitbox30
3
Extract the iOS platform
tar -xzf ~/Downloads/PitBox-iOS-Platform.tar.gz

Note: Adjust the path if you downloaded to a different location

4
Verify the folder was created
ls -la ios/

You should see folders like: App, App.xcodeproj, App.xcworkspace

5
Add to Git
git add ios/
6
Commit the changes
git commit -m "Add iOS platform for Capawesome Cloud builds"
7
Push to GitHub
git push origin main
8
Verify on GitHub

Visit https://github.com/fsumike/pitbox30

Confirm the ios/ folder is visible in your repository

✅ Next Step: Once the ios/ folder is pushed to GitHub, trigger a new build in Capawesome Cloud. The build will now find the iOS platform and proceed successfully!

What's Included

  • Native iOS project structure (Xcode workspace)
  • Capacitor iOS bridge configuration
  • CocoaPods Podfile for native dependencies
  • 13 Capacitor plugins configured
  • 1 Cordova plugin (cordova-plugin-purchase)

Troubleshooting

If extraction fails:

tar -xzf PitBox-iOS-Platform.tar.gz -C .

If Git won't add the files:

git rm -r --cached ios/
git add ios/
git commit -m "Re-add iOS platform"

To verify iOS folder is tracked:

git ls-files | grep ios/
