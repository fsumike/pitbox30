Enable pgcrypto Extension

Instructions:

  1. Download the SQL file below
  2. Go to your Supabase project dashboard
  3. Navigate to SQL Editor
  4. Create a new query
  5. Copy and paste the contents of the downloaded file
  6. Click Run
  7. Refresh your app
Download ENABLE_PGCRYPTO.sql

This will enable the pgcrypto extension which provides the gen_salt() function needed for secure PIN code hashing.
