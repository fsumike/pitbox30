📋 What to Do After Download

1
Extract the Archive

Right-click the downloaded file → Extract All... → Choose a location like C:\Users\YourName\Projects\

2
Open Command Prompt in the Project Folder

Navigate to the extracted folder, then:

  • Hold Shift + Right-click in the folder
  • Select "Open PowerShell window here" or "Open Command Prompt here"
3
Verify You're in the Right Place

Run this command:

git status

✅ Should show: "On branch main"

❌ If you see "fatal: not a git repository" - you're in the wrong folder!

4
Create GitHub Personal Access Token
  1. Go to: https://github.com/settings/tokens
  2. Click "Generate new token (classic)"
  3. Name: PitBox Upload
  4. Check the repo checkbox
  5. Click "Generate token"
  6. Copy the token immediately! (starts with ghp_...)
5
Create the Repository on GitHub (if not exists)
  1. Go to: https://github.com/new
  2. Repository name: pitbox-app
  3. Keep it Public or Private
  4. ⚠️ DO NOT initialize with README
  5. Click "Create repository"
6
Push to GitHub

In your terminal/PowerShell, run:

git push -u origin main

When prompted:

  • Username: fsumike
  • Password: [Paste your Personal Access Token]

🔧 Troubleshooting

Problem: "fatal: not a git repository"

→ You're not in the project folder. Use cd to navigate there.

Problem: "remote: Repository not found"

→ Create the repository on GitHub first (Step 5)

Problem: "Authentication failed"

→ Make sure you're using the Personal Access Token as the password, not your GitHub password

💡 Pro Tip: After the first successful push, you can bookmark the repository at https://github.com/fsumike/pitbox-app
